## 🚨 TROUBLESHOOTING

### Problem: "Application won't load"
**Solution:** 
- Check internet connection
- Verify https://lms.company.com is accessible
- Try different browser
- **Reference:** Backup screenshot of app

### Problem: "GitHub repo not accessible"
**Solution:**
- Verify manager has GitHub account
- Share repo via: https://github.com/company/leave-management-system
- Have desktop demo ready as backup

### Problem: "Dashboard not loading"
**Solution:**
- Refresh page (Ctrl+R)
- Check connection to monitoring service
- Have screenshot ready as backup

### Problem: "I forgot what to say next"
**Solution:**
- Glance at MANAGER_DEMO_SCRIPT.md
- It has exact talking points
- Or reference VISUAL_FLOW_DIAGRAMS.md for context

### Problem: "Manager asks technical question I don't know"
**Solution:**
- Say: "Great question, let me check" (buy time to look it up)
- Reference PROJECT_DEMO_GUIDE.md FAQ section
- If still unsure: "I'll follow up with that answer"
